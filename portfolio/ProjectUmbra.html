PROJECT UMBRA

Description

Project Umbra is a blackout top-down shooter. More information about the game will be disclosed in the future.


Controls

  • WASD - Move Character
  • R - Reload Weapon
  • F - Toggle Flashlight
  • Left Click - Shoot Gun
