PROJECT UMBRA

Description

Project Umbra is a blackout top-down shooter. More information about the game will be disclosed in the future.


Controls:

  • WASD - Move Character
  • R - Reload Weapon
  • F - Toggle Flashlight
  • Left Click - Shoot Gun

Team:

  • Noel Eugenio - Programmer, Artist
  • Jared Ritchie - Programmer, Designer
  • Mark Henry - Programmer, Designer
  • Spencer Cheung - Programmer, Sound Designer

Development Tools:

  • Unity
  • Autodesk Maya
